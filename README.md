# Chase CommonLib
A library of common functions and classes for .NET 6.0+.

## Installation
CommonLib is available on [NuGet](https://www.nuget.org/packages/Chase.CommonLib/).

.NET CLI:
```bash
dotnet add package Chase.CommonLib
```
Nuget:
```bash
Install-Package Chase.CommonLib
```
